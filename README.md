## Installation
### Using npm(ES6 module):
> **Note:** `w-batch` is mainly dependent on `json5`.

> **Note:** `w-batch` can be compiled into an executable file by using `sea`. 

```alias
npm i w-batch
```

> **Run for windows:**
```
path/to/wbat.exe path/to/setting.json
```
